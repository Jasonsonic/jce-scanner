# JCE Scanner V3.4 — 三个月低点加分版

## 新增规则

当前收盘价越接近过去约3个月（63个交易日）的最低价，获得额外加分。

计算方法：

```text
距离三个月低点
=
（当前收盘价 ÷ 过去63个交易日最低价 - 1）× 100%
```

评分：

- 距最低价不超过3%：+10分
- 高出3%～7%：+8分
- 高出7%～10%：+6分
- 高出10%～15%：+3分
- 高出超过15%：不加分

该加分同时作用于：

- 均线压缩建仓评分
- 连跌首阳反转评分

两套评分最终仍封顶100分。

## 保留的连跌首阳参数

- 最多向前检查15个交易日
- 最多忽略4天小幅上涨
- 小幅上涨不超过3%
- 整个阶段净跌至少8%
