# JCE Scanner V3 — 最新交易日建仓版

## 核心逻辑

- 只分析最新交易日，用于下一交易日开盘前筛选。
- MA5、MA8、MA13之间不要求任何排列顺序。
- 三条短期均线最好全部位于MA60上方。
- 四条均线越接近，得分越高。
- 最新收盘价越接近MA60，得分越高。
- 均线是否高于昨天、均线斜率，不参与核心评分。

## 总分100

- 四线压缩：40分
- 最新收盘价贴近MA60：30分
- 半年价格位置：10分
- 最近4日稳定性：10分
- 最近2日成交量：10分

如果MA5、MA8、MA13没有全部位于MA60上方，总分最高限制为69分。
