# JCE Scanner V3.3 — 15日下跌窗口版

## 本次参数调整

“连续下跌后首阳”规则改为：

- 最多向前检查15个交易日；
- 中间允许最多4个小幅上涨日；
- 小幅上涨定义为单日收盘涨幅不超过3%；
- 整个下跌阶段净跌至少8%；
- 阶段内至少包含3个真正下跌日；
- 最新交易日必须收阳；
- 最新收盘价必须高于前一交易日收盘价；
- 最新阳线涨幅超过10%时降低评级，避免追高。
